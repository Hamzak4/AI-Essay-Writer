AI Essay Writer — Full Stack AI Writing Platform

Built a full stack AI powered academic writing platform designed to assist users with multiple writing tasks including essay generation, paraphrasing, grammar correction, summarization, citation generation, essay outlining, and title brainstorming, all powered by Google Gemini API.

Tech Stack: React 18, Node.js, Express.js, MongoDB (with in memory fallback), JWT authentication, Google OAuth, bcrypt, Nodemailer

Key Contributions:

Designed and developed a secure RESTful API with user authentication, admin dashboard, and essay history tracking system
Integrated Google Gemini API to enable real time AI content generation across multiple writing tools
Built a responsive single page application using React with React Router, protected routes, and dynamic theme support
Implemented MongoDB database with automatic in memory fallback to ensure smooth deployment without configuration issues
Focused on performance, scalability, and clean user experience across all modules
